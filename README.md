## 🚀 快速开始

### 安装方法

1. 下载插件文件夹
2. 上传到 WordPress 站点：`wp-content/plugins/`
3. 在 WordPress 后台启用插件
4. 进入 WooCommerce → 设置 → 支付，配置参数

### 部署要求

- WordPress 5.0+
- WooCommerce 4.0+
- PHP 7.4+
