# 二维码生成说明

## 微信二维码

1. 打开微信 → 右上角"+" → "我的二维码"
2. 保存二维码图片
3. 重命名为 `qr-wechat.png`
4. 放到本目录下

## 在线生成工具

如果不想用个人微信，可以用企业微信或草料二维码：

- **草料二维码**：https://cli.im/url
  - 输入你的微信联系方式
  - 生成文本二维码
  - 下载 PNG 格式

## 替代方案

如果不想放二维码，可以修改 README.md：

```markdown
### 联系方式

- 📧 **邮箱**：your-email@example.com
- 💬 **微信**：your-wechat-id
- 🌐 **博客**：https://www.itbunan.xyz/
```
